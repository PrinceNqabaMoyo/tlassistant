## Introduction

* Exponential notation is a short way of writing the same number multiplied by itself or its inverse many times. We will now have a closer look at writing numbers using exponential notation. Exponents can also be called indices.
* Form: a^n, where a is the base and n is the exponent/index.
* For any real number a and natural number n, we can write a multiplied by itself n times as an.
* Basics to know:
1. a^n = a ├Ś a ├Ś a ├Ś ┬Ę ┬Ę ┬Ę ├Ś a (n times) (a Ōłł ŌäØ, n Ōłł ŌäĢ)
2. a^0 = 1 (a ŌēĀ 0 because 0^0 is undefined)
3. a^-n = 1/a^n (a ŌēĀ 0 because 1/0 is undefined)



Notice that we always write the final answer with positive exponents.



### Laws



* There are several laws we can use to make working with exponential numbers easier. Some of these laws might have been done in earlier grades, but we list all the laws here for easy reference:
1. a^m ├Ś a^n = a^(m+n)
2. a^m / a^n = a^(mŌłÆn)
3. (ab)^n = a^n b^n
4. (a/b)^n = a^n /b^n
5. (a^m)^n = a^ mn

where a > 0, b > 0 and m, n Ōłł ŌäØ.

\[2^(2n) ├Ś 4^n ├Ś 2]/16^n

## Exponential equations

Exponential equations have the unknown variable in the exponent. Here are some examples:

3^(x+1) = 9

5^t + 3 ├Ś 5^(tŌłÆ1) = 400



Solving exponential equations is simple: we need to apply the laws of exponents. This means that if we can write a single term with the same base on each side of the equation, we can equate the exponents.

Important: if a > 0 and a ŌēĀ 1



a^x = a^y

then x = y (same base)

Also notice that if a = 1, then x and y can be different.





### Examples

#### 1

Problem: Solve for x: 3^(x+1) = 9.



##### SOLUTION

###### Step 1 : Change the bases to prime numbers

3^(x+1) = 3^2



###### Step 2 : The bases are the same so we can equate exponents

x + 1 = 2

Ōł┤ x = 1



Note: to solve exponential equations, we use all the strategies for solving linear and quadratic equations.





#### 2\. 

Problem: Solve for t: 5^t + 3 ├Ś 5^(t+1) = 400.



##### SOLUTION

###### Step 1 : Rewrite the expression

5^t + 3(5^t ├Ś 5) = 400



###### Step 2 : Take out a common factor

5^t (1 + 15) = 400



###### Step 3 : Simplify

5^t(16) = 400

5^t = 25



###### Step 4 : Change the bases to prime numbers

5^t = 5^2



###### Step 5 : The bases are the same so we can equate exponents

Ōł┤ t = 2





#### 3\. 

Problem: Solve: p ŌłÆ 13p^1/2 + 36 = 0



##### SOLUTION

###### Step 1 : 

We notice that (p^1/2 )^2 = p so we can rewrite the equation as

(p^1/2 )^2 ŌłÆ 13p^1/2 + 36 = 0



###### Step 2 : Factorise as a trinomial

(p^1/2 ŌłÆ 9)(p^1/2 ŌłÆ 4) = 0



###### Step 3 : Solve to find both roots

p^ 1/2 ŌłÆ 9 = 0

p^ 1/2 = 9

(p^ 1/2)^2 = 9^2

p = 81



p^ 1/2 ŌłÆ 4 = 0

p^ 1/2 = 4

(p^ 1/2)^2 = 4^2

p = 16



Therefore p = 81 or p = 16.
