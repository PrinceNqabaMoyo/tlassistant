## Exponents and the Laws of Exponents

* Exponential notation is a short way of writing the same number multiplied by itself many times. We will now have a closer look at writing numbers using exponential notation. Exponents can also be called indices.
* Form: a^n, where a is the base and n is the exponent/index.
* For any real number a and natural number n, we can write a multiplied by itself n times as an.
* Basics to know: 

1. a^n = a √ó a √ó a √ó ¬∑ ¬∑ ¬∑ √ó a (n times) (a ‚àà ‚Ñù, n ‚àà ‚Ñï)
2. a^0 = 1 (a ‚âÝ 0 because 0^0 is undefined)
3. a^-n = 1/a^n (a ‚âÝ 0 because 1/0 is undefined)

Notice that we always write the final answer with positive exponents.

* There are several laws we can use to make working with exponential numbers easier. Some of these laws might have been done in earlier grades, but we list all the laws here for easy reference:

1. a^m √ó a^n = a^(m+n)
2. a^m / a^n = a^(m‚àín)
3. (ab)^n = a^n b^n
4. (a/b)^n = a^n /b^n
5. (a^m)^n = a^ mn

where a > 0, b > 0 and m, n ‚àà ‚Ñ§.

2\. Problem: Simplify:

\[2^(2n) √ó 4^n √ó 2]/16^n

SOLUTION

Step 1 : Change the bases to prime numbers

\[2^(2n) √ó 4n √ó 2]/16^n = \[2^(2n) √ó (2^2)^n √ó 2^1]/(2^4)^n

Step 2 : Simplify the exponents

= \[2^(2n) √ó 2^(2n) √ó 2^1]/2^(4n)

= \[2^(2n + 2n + 1)]/2^(4n)

= \[2^(4n + 1)]/2^(4n)

= 2^(4n + 1 - 4n)

= 2

3\. Problem: Simplify: \[5^(2x-1) 9^(x-2)]/15^(2x-3)

SOLUTION

Step 1 : Change the bases to prime numbers

\[5^(2x-1) 9^(x-2)]/15^(2x-3) = \[5^(2x‚àí1)(3^2)^(x-2)]/(5 √ó 3)^(2x - 3)

= \[5^(2x-1)3^(2x-4)] / \[5^(2x-3)3^(2x-3)]

Step 2 : Subtract the exponents (same base)

= 5^\[(2x‚àí1)‚àí(2x‚àí3)] √ó 3^\[(2x‚àí4)‚àí(2x‚àí3)]

= 5^(2x‚àí1‚àí2x+3) √ó 3^(2x‚àí4‚àí2x+3)

= 5^2 √ó 3^-1

Step 3 : Write the answer as a fraction

= 25/3

=8 1/3

Important: when working with exponents, all the laws of operation for algebra apply.

4\. Problem: Simplify: \[2^t ‚àí 2^(t‚àí2)]/\[3 √ó 2^t - 2^t]

SOLUTION

\[2^t ‚àí 2^(t‚àí2)]/\[3 √ó 2^t - 2^t] = \[2^t ‚àí (2^t√ó2^-2)]/\[3 √ó 2^t - 2^t]

5\. Problem: Simplify: (9^x - 1)/(3^x + 1)

SOLUTION

Step 1 : Change the bases to prime numbers

(9^x - 1)/(3^x + 1) = \[(3^2)^x - 1]/(3^x + 1)

### Rational exponents

We can also apply the exponential laws to expressions with rational exponents.

##### Examples

1. Problem: Simplify: 2x^(1/2) √ó 4x^(-1/2)

SOLUTION

2x^(1/2) √ó 4x^(-1/2) = 8x^(1/2 - 1/2)

= 8x^0

= 8(1)

= 8

2\. Problem: Simplify: (0,008)^ 1/3

SOLUTION

Step 1 : Write as a fraction and change the bases to prime numbers

0,008 = 8/1 000

= 2^3 / 10^3

= (2/10)^3

Step 2 : Therefore

(0,008)^1/3 = \[(2/10)^3]^(1/3)

= 2/10

= 1/5

### Exponential equations

Exponential equations have the unknown variable in the exponent. Here are some examples:

3^(x+1) = 9

5^t + 3 √ó 5^(t‚àí1) = 400

Solving exponential equations is simple: we need to apply the laws of exponents. This means that if we can write a single term with the same base on each side of the equation, we can equate the exponents.

Important: if a > 0 and a ‚âÝ 1

a^x = a^y

then x = y (same base)

Also notice that if a = 1, then x and y can be different.

##### Examples

1. Problem: Solve for x: 3^(x+1) = 9.

SOLUTION

Step 1 : Change the bases to prime numbers

3^(x+1) = 3^2

Step 2 : The bases are the same so we can equate exponents

x + 1 = 2

‚à¥ x = 1

Note: to solve exponential equations, we use all the strategies for solving linear and quadratic equations.

2\. Problem: Solve for t: 5^t + 3 √ó 5^(t+1) = 400.

SOLUTION

Step 1 : Rewrite the expression

5^t + 3(5^t √ó 5) = 400

Step 2 : Take out a common factor

5^t (1 + 15) = 400

Step 3 : Simplify

5^t(16) = 400

5^t = 25

Step 4 : Change the bases to prime numbers

5^t = 5^2

Step 5 : The bases are the same so we can equate exponents

‚à¥ t = 2

3\. Problem: Solve: p ‚àí 13p^1/2 + 36 = 0

SOLUTION

Step 1 : We notice that (p^1/2 )^2 = p so we can rewrite the equation as

(p^1/2 )^2 ‚àí 13p^1/2 + 36 = 0

Step 2 : Factorise as a trinomial

(p^1/2 ‚àí 9)(p^1/2 ‚àí 4) = 0

Step 3 : Solve to find both roots

p^ 1/2 ‚àí 9 = 0

p^ 1/2 = 9

(p^ 1/2)^2 = 9^2

p = 81

p^ 1/2 ‚àí 4 = 0

p^ 1/2 = 4

(p^ 1/2)^2 = 4^2

p = 16

Therefore p = 81 or p = 16.
