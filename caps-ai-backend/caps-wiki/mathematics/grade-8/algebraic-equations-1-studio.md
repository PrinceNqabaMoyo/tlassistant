### Setting up equations

An equation is a mathematical sentence that is true for some numbers but false for other numbers. The following are examples of equations:

x + 3= 11 and 2x = 8

x + 3 = 11 is true if x = 8, but false if x = 3.

When we look for a number or numbers that make an equation true, we say that we are solving the equation. For example, x = 4 is the solution of 2x = 8 because it makes 2x = 8 true. (Check: 2 × 4 = 8)
