# Template System Modularization Plan
# Geometry Studio - Scalable Template Architecture for Grades 7-12

## 🎯 **Objective**
Transform the monolithic `question_templates.py` (5,201 lines) into a scalable, modular architecture that can support Grades 7-12 geometry content without becoming unmanageable.

## 📊 **Current State Analysis**

### **Current Problems:**
- Single file with 5,201 lines (Grade 7 only)
- Projected 30,000+ lines across all grades
- Difficult to maintain and extend
- No separation of concerns by grade level
- Hard to load only specific grade content
- Team collaboration challenges

### **Current Template Count:**
- Grade 7: 360 templates (60 per shape × 6 shapes)
- Projected Grades 7-12: 2,160 templates
- Estimated file size: 30,000+ lines

## 🔧 **Technical Implementation Details**

### **Base Classes:**
```python
# template_base.py
from abc import ABC, abstractmethod
from typing import Dict, List, Optional
from ..quiz_models import QuestionTemplate, DifficultyLevel

class GradeTemplateModule(ABC):
    """Abstract base class for grade-specific template modules"""
    
    def __init__(self, grade: int):
        self.grade = grade
        self.templates = self._initialize_templates()
    
    @abstractmethod
    def _initialize_templates(self) -> Dict[str, Dict[str, List[QuestionTemplate]]]:
        """Initialize templates for this grade"""
        pass
    
    @abstractmethod
    def get_templates(self, shape: str, difficulty: str) -> List[QuestionTemplate]:
        """Get templates for specific shape and difficulty"""
        pass
    
    @abstractmethod
    def get_available_shapes(self) -> List[str]:
        """Get list of available shapes for this grade"""
        pass
    
    @abstractmethod
    def get_available_difficulties(self) -> List[str]:
        """Get list of available difficulties for this grade"""
        pass
```

### **Registry System:**
```python
# template_registry.py
class TemplateRegistry:
    """Central registry for all grade template modules"""
    
    def __init__(self):
        self.grades: Dict[int, GradeTemplateModule] = {}
        self._load_available_grades()
    
    def register_grade(self, grade: int, module: GradeTemplateModule):
        """Register a grade module"""
        self.grades[grade] = module
    
    def get_templates(self, grade: int, shape: str, difficulty: str) -> List[QuestionTemplate]:
        """Get templates for specific grade, shape, and difficulty"""
        if grade not in self.grades:
            raise ValueError(f"Grade {grade} not available")
        return self.grades[grade].get_templates(shape, difficulty)
    
    def get_available_grades(self) -> List[int]:
        """Get list of available grades"""
        return sorted(self.grades.keys())
```

### **Dynamic Loading:**
```python
# template_loader.py
class TemplateLoader:
    """Dynamic template loading system"""
    
    def __init__(self):
        self.registry = TemplateRegistry()
        self._load_grade_modules()
    
    def _load_grade_modules(self):
        """Dynamically load all available grade modules"""
        import importlib
        import os
        
        templates_dir = os.path.dirname(__file__)
        for item in os.listdir(templates_dir):
            if item.startswith('grade') and os.path.isdir(os.path.join(templates_dir, item)):
                grade_num = int(item.replace('grade', ''))
                try:
                    module = importlib.import_module(f'.{item}', package='templates')
                    if hasattr(module, 'GradeTemplateModule'):
                        self.registry.register_grade(grade_num, module.GradeTemplateModule(grade_num))
                except ImportError as e:
                    print(f"Warning: Could not load grade {grade_num}: {e}")
```
